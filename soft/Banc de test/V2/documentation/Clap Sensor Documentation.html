Clap Sensor

The Clap Sensor sample project implements a clap sensor that detects claps and displays the intensity of the claps using the NI myRIO onboard LEDs.

System Requirements

Development System

  • LabVIEW Full or Professional Development System
  • LabVIEW myRIO Toolkit
  • LabVIEW Real-Time Module

Hardware

  • NI myRIO
  • A microphone

Overview

This sample project uses a Flat Sequence structure that executes the following three frames from left to right:

  1. Initialize—Initializes a Boolean array that represents the four onboard LEDs and opens a reference to the audio input channel.
  2. Acquire and process data—Acquires data from the audio input channel and processes the data for presentation.
  3. Close—Closes the reference to the audio input channel and resets the hardware output values to zero before the application exits.

Instead of using a While Loop, this sample project uses the Timed Loop to ensure precise timing for data acquisition. The Timed Loop executes the following steps on the NI myRIO RT target:

  1. Samples the audio input from the microphone and constructs a waveform.
  2. Detects peaks corresponding to claps.
  3. Turns on LEDs to display the intensity.

Running This Sample Project

Before you run this sample project, connect the microphone to the Audio In connector on the NI myRIO-1900 or the A/AI0 channel on the NI myRIO-1950.

Complete the following steps to run this sample project:

  1. In the Project Explorer window, open and run Main.vi.
  2. Observe the LED Status indicator for the intensity of the audio input.
  3. Click Stop to stop the application.
